JAXB

JAXB Component

The JAXB data format provides JAXB support. Camel supports unmarshalling XML data to JAXB annotated classes and marshalling from classes to XML. For more information, see JAXB.

Camel on EAP deployment

This component is supported by the Camel on EAP (Wildfly Camel) framework, which offers a simplified deployment model on the Red Hat JBoss Enterprise Application Platform (JBoss EAP) container.
