Disconnecting two patterns

Overview

As you develop and update a route, you may need to disconnect some nodes to insert additional patterns. The Remove option on the connector context menu makes this easy to do.

Important

If you leave a disconnected node unconnected, the tooling will reconnect it to a node when the route is reloaded in the route editor or when the context file is reopened, even if you saved the change before switching views or closing the routing context file.

Procedure

To disconnect two nodes:

  1. Select the connector you want to delete.

  2. Right-click it to open the context menu.

  3. Select Remove.

The connector between the nodes is removed.
