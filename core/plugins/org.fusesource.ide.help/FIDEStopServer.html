Stopping a Server

Overview

You can shut down a server in one of two ways:

  • from Servers panel

  • from the server's remote console in Terminal view

Using Servers panel

To stop a server:

  1. In Servers panel, select the server you want to stop.

  2. Click .

Using the remote console

To stop a server:

  1. Open the Terminal view hosting the server's remote console.

  2. Type CTRL+D.
